# 校园闲置物品租售系统

## 下载地址
https://bysj.bsitem.com/#/home/goodDetail?goodId=11944

## 技术介绍
Java语言 + ssm框架/springboot框架（后端）+ Vue前端框架（Web）+ Mysql(数据库)

## 开发工具
IDEA或者MyEclipse + Navicat(mysql数据库可视化工具) + Tomcat

## 项目视频演示
- [演示视频1](https://graduation-images.oss-cn-beijing.aliyuncs.com/videos/56%E5%A5%97-1-springboot%E5%BD%95%E5%83%8F/11944_springboot777%E6%A0%A1%E5%9B%AD%E9%97%B2%E7%BD%AE%E7%89%A9%E5%93%81%E7%A7%9F%E5%94%AE%E7%B3%BB%E7%BB%9F.mp4)

## 项目资料包含
1. 前后端源代码
2. 开题报告（任务书）+ 配套论文（一万两千字以上）+ 答辩PPT
3. 远程控屏调试（向日葵或者toDesk控屏）
4. 一键启动项目（双击运行演示，无需搭建复杂环境）
5. 项目讲解视频教程
